## How to Update This Changelog

When contributing, please update this file following these guidelines:

1. Add new entries under the `[Unreleased]` section
2. Use the appropriate category (Added, Changed, Fixed, etc.)
3. Write clear, concise descriptions
4. Include issue/MR references when applicable
5. When releasing, move items from Unreleased to a new version section

Example:
```markdown
## [Unreleased]

### Added
- Voice input support for chat interface (#123)

### Fixed
- Resolved memory leak in conversation history (#124)
```
